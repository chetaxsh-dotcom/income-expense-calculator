
# Income Expense Calculator

A simple and responsive Income Expense Calculator built using HTML, CSS, and JavaScript.  
This application allows users to manage their income and expenses and view their financial summary.

## Features
- Add income and expense entries
- Edit and delete entries
- Filter entries (All / Income / Expense)
- View total income, total expense, and balance
- Data persistence using Local Storage
- Responsive and clean UI

## Tech Stack
- HTML
- CSS
- JavaScript

## How to Run
1. Clone the repository
2. Open `index.html` in a browser

## Deployment
This project is deployed using Netlify.
